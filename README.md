# Pixel Apocalypse — Godot Project Skeleton

Generated by /plan-eng-review from design doc.

## Project Status

这是代码骨架，不是完整游戏。第一天你需要:
1. 在 Godot 4.x 中打开此项目
2. 创建缺失的场景文件 (.tscn)
3. 绑定资源和音乐文件

## File Map

```
project.godot          — 项目配置 (320x180 viewport, pixel perfect)
scenes/                — 场景文件 (需要用 Godot 编辑器创建 .tscn)
scripts/
  main.gd              — 游戏状态机 (EXPLORE/COUNTDOWN/DEFEND/PHASE_END)
  player.gd             — 角色 (WASD, 射击, E 互动, 100HP)
  zombie.gd             — 僵尸 (追踪玩家, 接触伤害)
  fog_of_war.gd        — 迷雾系统 (Area2D reveal, 永久揭示)
  wave_manager.gd      — 波次管理 (3 波递增, 定时触发)
  shelter.gd           — 庇护所 (进入触发, 门锁状态)
  music_manager.gd     — 音乐切换 (A/B 交叉淡入淡出, 2 秒)
  resource_pickup.gd   — 资源道具 (可见/隐形, E 发现)
resources/
  sprites/             — 精灵图 (Kenney CC0 过渡资源)
  audio/                — 音乐文件 (CC0, 待填入路径)
```

## State Machine

```
EXPLORE ─[进入庇护所]→ COUNTDOWN ─[倒计时结束]→ DEFEND ─[3波清完]→ PHASE_END
```

## Key Implementation Notes

1. **Pixel Perfect**: viewport 320x180, stretch mode=viewport, texture filter=off
2. **Parallax**: 使用 ParallaxLayer + ParallaxMaterial, scroll_offset 控制
3. **Fog**: TileMapLayer, set_cell(coords, -1) 清除揭示迷雾
4. **Wave Manager**: zombie_scene 需要预先在编辑器中配置为 zombie.tscn
5. **Music**: _track_paths 字典需要填入实际 .ogg 文件路径

## Controls

|-----|--------|

| E | Interact (discover hidden resources) |
| LMB | Shoot (DEFEND mode only) |
